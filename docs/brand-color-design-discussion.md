## 二、竞品一体化设计共同规律

### 2.1 两种一体化策略

| 策略 | 做法 | 使用者 |
|------|------|--------|
| 策略A: 透明导航栏 | 导航栏透明 → 大Banner铺满上半屏 → 白色内容区 | 霸王茶姬、塔斯汀、蜜雪冰城 |
| 策略B: 同色导航栏 | 导航栏与页面背景完全同色 → 白色卡片浮在统一背景上 | 天工个人中心页 |

### 2.2 共同特征

1. **颜色只有2层** — 品牌色 + 白色，不会出现第三种颜色
2. **导航栏不是独立色块** — 要么透明，要么与下方同色
3. **区块之间无结构性分割** — 不用圆角+边框+阴影+margin来分隔相邻区块
4. **上半屏是一整块视觉区域** — 从状态栏到会员信息区之间没有任何分割

### 2.3 天工抽奖页当前问题

天工抽奖页导航栏是 `#667eea`（紫色），页面主体渐变是 `#f7931e`（橙色），出现了第三种颜色，打破了2层配色规律，导致视觉分割。

## 四、文创品牌配色方案（候选）

天工是文创品牌，核心是**创意、文化、审美、个性**。以下是三个候选方向：

### 方案A：莫兰迪色系 — 艺术馆调性

```
主色:   #8d7b6a 灰棕
辅色:   #d4c5b2 浅米
背景:   #f5f0ea 象牙白
点缀:   #c17f59 陶土橙（按钮、重点元素）
文字:   #4a4a4a 深灰
```

| 维度 | 评价 |
|------|------|
| 品牌感受 | 高级、克制、有审美品位，像走进一家设计师店铺 |
| 用户心理 | "这个品牌很有品位，用它的东西显得我有审美" |
| 适合调性 | 匠心、手作、设计感、材质感 |
| 参考品牌 | 无印良品、方所书店、诚品生活 |
| 与文创契合度 | ★★★★★ |

**优势**: 高级感最强，不会过时，适合长期品牌建设
**劣势**: 视觉冲击力弱，抽奖等活跃场景可能显得"冷淡"

---

### 方案B：雅紫+金 — 创意潮牌调性

```
主色:   #667eea 靛蓝紫 / #7c3aed 创意紫
辅色:   #f0edff 浅紫白
背景:   #ffffff 纯白
点缀:   #f7931e 金橙（积分、奖励等价值感元素）
文字:   #333333 深灰
```

| 维度 | 评价 |
|------|------|
| 品牌感受 | 有创意、有个性、潮流但不浮躁 |
| 用户心理 | "这个品牌很酷，很有创意，我愿意收藏它的东西" |
| 适合调性 | 创意、潮流、IP联名、盲盒 |
| 参考品牌 | 得物、泡泡玛特、故宫文创 |
| 与文创契合度 | ★★★★☆ |

**优势**: 年轻化、有辨识度，抽奖/游戏化场景天然适配
**劣势**: 紫色如果用不好容易显得廉价，需要精细把控

## 五、配色方案选择决策树

```
天工的品牌调性是什么？
│
├─ 国潮文创（传统文化 + 现代设计）
│  └─ → 方案C: 中式文化色（故宫红/藏蓝 + 宣纸黄 + 古金）
│
├─ 现代潮流文创（IP、盲盒、创意设计）
│  └─ → 方案B: 雅紫+金（靛蓝紫 + 浅紫白 + 金橙点缀）
│
├─ 匠心手作文创（手工、设计师、材质）
│  └─ → 方案A: 莫兰迪色系（灰棕 + 浅米 + 陶土橙点缀）
│
└─ 还没确定 / 混合调性
   └─ → 建议先确定品牌调性，再选配色
       （配色是品牌调性的视觉表达，调性不清晰时选色容易反复）
```
