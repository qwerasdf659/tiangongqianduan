**后端不存在导致"积分能显示但钻石/红水晶碎片不能显示"的差异。**

### 需要前端排查的方向

由于积分图片能显示证明图片代理本身工作正常，问题可能出在：

1. **微信小程序图片缓存**：真机可能缓存了之前加载失败的图片（旧版 API 返回不同 URL 或无 URL 时的失败状态）。请在真机上 **长按小程序 → 删除 → 重新搜索进入**（不只是清缓存，要完全删除重进）
2. **前端降级链是否干扰了图片 URL**：检查 `addPrizeIcon()` 中第 2 级降级（`material_asset_code → 本地 WebP`）是否覆盖了第 1 级（`image.url`）。如果 `prize.image.url` 存在但同时 `prize.material_asset_code` 也存在，是否存在某些条件下第 2 级优先于第 1 级？
3. **vConsole 抓包**：在真机调试时打开 vConsole → Network 面板，查看钻石/红水晶碎片图片的请求状态和错误信息
