## 五、前端数据安全说明

### 当前安全措施

| 措施 | 说明 |
|------|------|
| HTTPS强制 | 测试/生产环境API地址均为 https:// |
| Token不拼URL | Socket.IO通过auth选项传递Token，不拼在URL上 |
