# 交易市场合规运营指南

**适用平台**：天工商户数字化营销服务平台
**更新日期**：2026年6月2日

---

## 一、交易市场定性

本平台交易市场是**封闭虚拟经济体内的物品流转服务**，所有交易以虚拟资产（星石、源晶碎片等）结算，不涉及任何法定货币。

法律性质等同于游戏内交易所（如Steam市场、梦幻西游藏宝阁），不构成金融活动。

---

## 二、四条绝对红线

| 红线 | 说明 | 违反后果 |
|------|------|---------|
| 永远不开法币充值 | 用户不能花人民币购买星石/材料/任何虚拟资产 | 一旦开放，星石立即变成代币，涉嫌非法发行虚拟货币 |
| 永远不开法币提现 | 用户不能把虚拟资产换成人民币 | 一旦开放，整个系统变成赌博/非法集资平台 |
| 高价值实物不可交易 | 可兑换高价值实物的物品/凭证必须设为 `is_tradable=0` | 否则形成"抽奖→材料→交易→兑换高价实物"的变现链条 |
| 永远不做官方回收 | 平台不以任何形式（人民币/星石/积分）回收用户已获得的物品或已兑换的商品 | 否则形成"虚拟资产↔商品"的价值闭环，星石变成有保值功能的代币 |

## 三、可交易物品范围

### 允许在交易市场流通的

- 星石（star_stone）
- 红/橙/黄/绿/蓝/紫源晶碎片（shard）
- 红/橙/黄/绿/蓝/紫源晶宝石（gem）
- 抽奖获得的虚拟物品（背包道具，`is_tradable=1`）
- DIY作品（虚拟设计稿）
- 其他平台发行的虚拟道具

### 禁止在交易市场流通的

- 积分（points）— 硬编码黑名单
- 预算积分（budget_points）— 硬编码黑名单
- 可兑换高价值实物的物品/凭证（`is_tradable=0`）
- 会员等级权益兑换获得的实物凭证
- 任何有明确人民币市场价格对标的物品

### 判断标准

一个物品是否可以进入交易市场，用这个问题判断：

> "用户拿到这个东西后，能不能在系统外把它变成人民币？"

- 答案是"不能" → 可以交易
- 答案是"能"或"间接能" → 禁止交易

## 四、定价管控规则

| 规则 | 说明 |
|------|------|
| 定价币种白名单 | 仅允许 star_stone 和 red_core_shard 作为计价货币 |
| 价格区间限制 | 每种币种设有最低价和最高价，防止异常定价 |
| 价格偏离监控 | 挂单价格偏离近期成交均价超过一定比例时，触发人工审核 |

---

## 五、反场外交易监控

### 必须监控的异常行为

| 异常模式 | 触发条件 | 处理方式 |
|---------|---------|---------|
| 异常低价挂单 | 价格低于市场均价50%以上 | 自动拦截 + 人工审核 |
| 固定对手交易 | 同两个用户7天内互相交易超过5次 | 标记 + 告警 + 人工审核 |
| 新号大额交易 | 注册7天内单笔交易超过1000星石 | 限制交易额度 |
| 批量同价挂单 | 同一用户短时间内挂出大量相同价格的商品 | 频率限制 |
| 高频撤单 | 单日撤单超过N次 | 限制当日挂单权限 |

### 处理措施升级路径

```
第一次触发 → 系统警告
第二次触发 → 限制交易功能7天
第三次触发 → 永久封禁交易市场权限
情节严重 → 封禁账号 + 扣除全部虚拟资产
```

| VIP用户 | 20000 | 30 |

## 七、手续费策略

| 计价币种 | 手续费率 | 最低手续费 | 收费对象 |
|---------|---------|-----------|---------|
| star_stone（星石） | 分档制（3%-8%） | 1星石 | 卖方 |
| red_core_shard（红源晶碎片） | 5% | 1碎片 | 卖方 |

手续费的作用：
- 平台收入来源之一
- 星石消耗出口（防通胀）
- 增加场外交易的"摩擦成本"（场外交易没有平台保障，但也省不了多少手续费）

## 八、高价值饰品上架时的操作清单

当你未来在兑换商城加入价值几千元的饰品时，必须执行以下操作：

- [ ] 对应的 `item_template` 设置 `is_tradable = 0`
- [ ] 兑换获得的物品实例（items表）继承 `is_tradable = 0`
- [ ] 兑换商城页面不标注商品的人民币市场价格
- [ ] 宣传话术使用"尊享礼遇"、"限定臻品"，不使用"价值XX元"
- [ ] 兑换条件设置为"等级+虚拟资产+其他条件"的组合，不是纯星石直接兑换
- [ ] 用户协议中已声明"虚拟资产不具有现金价值"

---

## 九、用户协议中必须包含的交易市场条款

以下条款已写入《用户服务协议》第五章，确保持续有效：

1. "交易市场为平台内虚拟物品流转服务，所有交易以虚拟资产结算，不涉及法定货币"
2. "严禁通过任何方式将虚拟资产兑换为现金或进行场外交易"
3. "平台有权对异常交易行为进行监控、限制或封禁"
4. "虚拟资产不具有法定货币等价物属性，不可兑换为人民币"

## 十、定期自查清单

每月执行一次以下检查：

- [ ] 确认系统中不存在法币充值/提现入口
- [ ] 确认 points 和 budget_points 仍在交易黑名单中
- [ ] 检查是否有新上架的高价值兑换商品未设置 `is_tradable=0`
- [ ] 审查交易市场异常交易告警记录
- [ ] 确认抽奖奖品池中无 `physical` 类型的活跃奖品
- [ ] 确认广告系统未接入外部商家的星石支付

## 十一、合规安全总结

你的交易市场合法运营的核心逻辑：

```
封闭系统（无法币入口/出口）
  + 交易物无外部变现渠道
  + 高价值实物不进入交易流通
  + 反场外交易监控
  + 用户协议明确声明
  = 合法的虚拟物品流转服务
```

只要这五个条件同时满足，交易市场就能长期安全运营。任何一个条件被打破，都需要立即评估风险并修复。
