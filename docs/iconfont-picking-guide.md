# iconfont.cn 图标挑选清单

## 操作说明

1. 打开 https://www.iconfont.cn
2. 搜索下表中的"搜索关键词"
3. 筛选风格：**填充/面性**（不要线性）
4. 尽量选同一个作者或同一套图标库的，保证风格统一
5. 点击图标加入购物车，然后添加到项目
6. 在项目中把图标名改为"项目中的名称"列对应的值

## 项目设置

- Font Family：`tiangong-icons`
- 前缀：`icon-`
- 字体格式：woff2

## 挑选技巧

1. **风格统一最重要** — 宁可全部选"还行"的同一套，也不要东拼西凑"各自精致"的
2. **搜索时加"填充"** — 比如搜"铃铛 填充"或筛选面性风格
3. **推荐找整套图标库** — 搜索"通用图标"或"电商图标"，找一个 50+ 图标的库，大概率能覆盖大部分需求
4. **图标大小统一** — 都选 1024x1024 或 128x128 的，不要混用不同画布尺寸
5. **游戏类图标单独找** — 搜"游戏 图标"或"抽奖 图标"找专门的游戏图标套装

## 推荐搜索策略

先搜这些关键词找整套的：
- "电商图标 填充"
- "金融图标 填充"  
- "游戏图标 填充"
- "通用图标 圆润"

如果找到一套风格满意的，直接把整套加入项目，然后删掉不需要的。
