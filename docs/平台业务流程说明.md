# 天工平台业务流程说明

**平台定位**：商户数字化营销服务平台（MarTech）
**运营主体**：东莞市天工智造金属有限责任公司
**更新日期**：2026年6月2日

---

## 一、商业模式总览

天工平台是一个连接**消费者、商户、平台**三方的数字化营销服务系统。

```
商户支付推广服务费（人民币）
       ↓
平台提供引流和会员运营服务
       ↓
用户到商户消费 → 获得积分奖励
       ↓
用户在平台内参与游戏化互动（抽奖、合成、交易、DIY）
       ↓
用户消耗虚拟资产兑换实物商品或享受会员权益
```

**收入来源**：商户推广返点/服务费（B2B人民币结算）
**成本支出**：兑换商品采购、平台运营、技术开发

## 二、核心业务流程

### 2.1 用户获取积分

```
用户到合作商户消费（如餐饮消费150元）
       ↓
商户扫码确认消费（动态二维码核销）
       ↓
平台记录消费金额，按规则发放积分（如1元=1积分）
       ↓
积分进入用户账户（asset_code: points）
```

### 2.2 积分抽奖获得虚拟资产

```
用户消耗积分参与抽奖活动（如消耗30积分/次）
       ↓
系统按概率算法（高5%/中15%/低80%）随机出奖
       ↓
奖品为不同数量的虚拟材料：
  - 星石（star_stone）— 主流通货币
  - 红/橙/黄/绿/蓝/紫源晶碎片（shard）— 材料
  - 红源晶宝石（gem）— 高级材料
  - 积分返还（points）— 保底奖
       ↓
虚拟资产进入用户账户
```

### 2.3 材料合成与转换

```
用户持有低级材料（如10个红源晶碎片）
       ↓
通过资产转换规则合成高级材料（10红碎片 → 1星石）
       ↓
不同颜色材料有不同的转换比例：
  - 红碎片：10个 → 1星石
  - 橙碎片：10个 → 1星石
  - 黄碎片：5个 → 1星石
  - 蓝碎片：2个 → 1星石
  - 紫碎片：1个 → 1星石
  - 红源晶（gem）：1个 → 2星石
```

### 2.4 交易市场（用户间流转）

```
卖方：
  用户将虚拟物品/材料挂单出售
       ↓
  设定价格（以星石或红源晶碎片计价）
       ↓
  系统冻结卖方的物品/材料

买方：
  浏览交易市场，选择心仪商品
       ↓
  点击购买，系统冻结买方的虚拟资产
       ↓
  交易完成：
    - 物品/材料转移给买方
    - 虚拟资产（扣除5%手续费）转移给卖方
    - 手续费归平台（星石销毁）
```

### 2.5 兑换商城

```
用户进入兑换商城
       ↓
选择商品（实物商品，如衣服、饰品、日用品）
       ↓
支付虚拟资产（如10红源晶碎片兑换一件衣服）
       ↓
生成兑换订单
       ↓
平台审核 → 发货 → 用户收货确认
```

### 2.6 DIY定制

```
用户选择DIY模板
       ↓
使用平台提供的设计工具进行个性化创作
       ↓
消耗虚拟材料（星石+源晶碎片）完成制作
       ↓
生成虚拟作品（存入背包）
       ↓
可在交易市场出售（虚拟设计稿）或申请实物制作
```

### 2.7 广告系统

```
用户推广（星石支付）：
  用户消耗星石 → 购买挂单置顶/曝光加速等增值服务

商户广告（人民币支付）：
  商户支付人民币 → 购买平台广告位（首页轮播、弹窗等）
  平台可额外用星石激励用户互动（如"看广告得星石"活动）
```

### 2.8 会员等级体系

```
用户通过以下行为积累等级：
  - 累计消费金额
  - 平台活跃度（登录、互动）
  - 完成特定任务
       ↓
等级提升后解锁对应权益：
  - 专属标识/头像框
  - 交易市场手续费折扣
  - 更多挂单位
  - 年度礼品兑换资格（高等级专属）
  - 限定活动参与资格
```

## 三、资金流与虚拟资产流

### 资金流（人民币，B2B）

```
商户 →（推广服务费/返点）→ 平台
商户 →（广告费）→ 平台
```

平台收入全部为人民币，来源于商户端。用户端不涉及任何人民币交易。

### 虚拟资产流（封闭系统）

```
平台 →（积分发放）→ 用户    [消费奖励]
平台 →（材料/星石发放）→ 用户  [抽奖奖励]
用户 →（积分消耗）→ 平台    [参与抽奖]
用户 ←→ 用户              [交易市场流转]
用户 →（虚拟资产消耗）→ 平台  [兑换商品/DIY/广告/手续费]
```

**关键特征**：虚拟资产只在"平台↔用户"和"用户↔用户"之间流动，永远不与法币发生兑换关系。

## 六、合规架构

```
                    ┌─────────────────────────┐
                    │      法币世界（外部）      │
                    │                         │
                    │  商户 →（人民币）→ 平台   │
                    │  （推广费/广告费）         │
                    └────────────┬────────────┘
                                 │
                          ═══════╪═══════  防火墙（无法币入口/出口）
                                 │
                    ┌────────────▼────────────┐
                    │    虚拟经济体（内部封闭）    │
                    │                         │
                    │  积分 → 抽奖 → 材料/星石  │
                    │       ↕                 │
                    │  交易市场（用户间流转）     │
                    │       ↓                 │
                    │  兑换商城 / DIY / 广告    │
                    │  （虚拟资产消耗终点）       │
                    └─────────────────────────┘
```

**核心安全机制**：
- 法币与虚拟资产之间有不可逾越的防火墙
- 虚拟资产只能在封闭系统内流转和消耗
- 积分（points）硬编码禁止交易
- 高价值实物设为不可交易
- 平台永不回收用户已获得的物品/商品

## 七、各方角色与关系

| 角色 | 在系统中的行为 | 与平台的关系 |
|------|-------------|------------|
| 消费者（用户） | 消费获积分、抽奖、交易、兑换、DIY | 使用平台服务，不支付人民币 |
| 商户 | 提供消费场景、支付推广费 | B2B合作，人民币结算 |
| 平台 | 提供技术系统、运营服务、发放虚拟资产 | 营销科技服务商 |

## 八、业务定性

本平台业务定性为：**基于消费行为的CPS联盟营销 + 游戏化用户忠诚度计划**

等同于：
- 肯德基/麦当劳积分商城（消费得积分换礼品）
- 航空公司里程计划（飞行得里程换机票）
- 京东京豆体系（购物得京豆抵现）

加上游戏化元素：
- 抽奖互动（趣味性）
- 材料收集与合成（成就感）
- 用户间交易市场（社交性）
- DIY定制（创造性）
