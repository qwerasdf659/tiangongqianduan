# 🎮 扭蛋机 - 80年代游戏厅复古怀旧风格

## 📅 版本信息
- **风格**: 复古怀旧 Retro Arcade
- **更新时间**: 2026年2月14日
- **设计灵感**: 80年代街机厅、怀旧玩具扭蛋机
- **配色主题**: 木质+黄铜+琥珀玻璃

---

## 🎨 设计特色

### 1. **配色方案**
```
主色调：
- 木质棕 #8b4513, #654321
- 黄铜色 #daa520, #cd7f32, #b8860b
- 琥珀玻璃 rgba(255, 228, 181, 0.6)
- 复古黄 #ffeb3b (提示文字)

辅助色：
- 深褐背景 #2c1810, #3d2415
- 黑色描边 #000000
- 暖光晕 rgba(255, 170, 51, 0.15)
```

### 2. **材质质感**
- **机身框架**: 木质纹理 + 条纹装饰
- **圆顶玻璃**: 有色玻璃（琥珀色）+ 柔和反光
- **黄铜装饰**: 金属光泽 + 绿色铜锈
- **手柄旋钮**: 木质 + 木纹纹理
- **扭蛋球**: 磨砂塑料 + 暖色高光

### 3. **装饰元素**
- 木质边框（圆角12-20rpx）
- 黄铜装饰条（底部，带锈迹）
- 手柄臂金属磨损纹理
- 像素风格文字（Courier New）
- 黑色粗描边效果

### 4. **动画风格**
```
机械感动画（保留）：
✅ 手柄转动 - 机械卡嗒感
✅ 扭蛋球缓慢浮动 - 悠闲节奏
✅ 简单上浮粒子 - 8-bit风格

现代科技感（移除）：
❌ 霓虹光晕旋转
❌ 能量蓄积条
❌ 3D螺旋粒子
❌ 动态光束
❌ 多层脉冲光环
```

## 🕹️ 设计理念

### **怀旧游戏厅氛围**
- 80年代街机厅的木质装饰
- 黄铜色金属零件的年代感
- 琥珀色玻璃的暖光效果
- 像素字体的复古韵味

### **机械玩具感**
- 简洁的机械动作
- 木质手柄的触感暗示
- 扭蛋球缓慢浮动的真实感
- 单次震动反馈的质朴感

### **视觉做旧效果**
- 木纹纹理（条纹装饰）
- 黄铜锈迹（绿色氧化）
- 金属磨损（划痕暗示）
- 暖色光晕（老式灯光）

## 🔧 自定义建议

### **如需更复古**
- 添加划痕纹理贴图
- 增加木质纹理深度
- 更多铜锈效果
- 降低整体明度

### **如需更现代一点**
- 减少木纹纹理
- 提升玻璃透明度
- 使用更明亮的黄铜色
- 添加少量光效

### **配色替代方案**
```scss
// 深色主题
$wood-brown: #3d2415;
$brass-gold: #b8860b;

// 明亮主题
$wood-brown: #d2691e;
$brass-gold: #ffd700;
```
