**创建日期**: 2026-02-14

---

## 🎯 设计目标

稀有度光效作为**增强层**，硬编码样式作为**保底层**，确保在任何情况下用户都能看到美观的界面。

---

## 📊 三种场景处理

### 场景1：正常使用稀有度光效 ✨

**条件**：
- 后端返回 `rarity` 字段（common/rare/epic/legendary）
- 后端配置 `rarity_effects_enabled: true`

**效果**：
```
<rarity-effects rarity="epic" enabled="true">
  <view class="prize-card-bg">
    <!-- 奖品内容 -->
  </view>
</rarity-effects>

显示：橙色光环 + 闪烁动画 + 内部硬编码的渐变背景
```

---

### 场景2：后端未返回 rarity 字段 🔄

**条件**：
- 后端未返回 `rarity` 字段（或返回无效值）
- 后端配置 `rarity_effects_enabled: true`

**效果**：
```
<rarity-effects rarity="undefined" enabled="true">
  <!-- 自动降级为 rarity="common" -->
</rarity-effects>

显示：蓝色边框（common 默认样式）+ 内部硬编码的渐变背景
```

**代码实现**：
```javascript
// rarity-effects.ts
const validRarities = ['common', 'rare', 'epic', 'legendary']
const safeRarity = validRarities.includes(rarity) ? rarity : 'common'
```

---

### 场景3：关闭稀有度光效（保底机制） 🛡️

**条件**：
- 后端配置 `rarity_effects_enabled: false`
- 或运营选择不使用稀有度光效

**效果**：
```
<rarity-effects rarity="epic" enabled="false">
  <view class="prize-card-bg">
    <!-- 奖品内容 -->
  </view>
</rarity-effects>

显示：只显示内部硬编码的渐变背景和阴影（rarity-effects 完全透明）
```

**代码实现**：
```javascript
// rarity-effects.ts
this.setData({
  rarityClass: enabled ? `rarity--${safeRarity}` : ''  // ← enabled=false 时为空字符串
})
```

```scss
// rarity-effects.scss
.rarity-effects {
  position: relative;
  border-radius: 16rpx;
  overflow: visible;
  /* 默认无任何样式，完全透明容器 */
}

/* 只有添加了 rarity--xxx class 才有样式 */
.rarity--common { border: 2rpx solid #3498db; }
.rarity--rare { border: 2rpx solid #9b59b6; animation: rarity-pulse 2s infinite; }
.rarity--epic { border: 2rpx solid #e67e22; animation: rarity-flicker 1.5s infinite; }
.rarity--legendary { border: 2rpx solid #f1c40f; box-shadow: ...; }
```

## 🏗️ 多层次视觉保障

```
第1层（最底层）：硬编码样式 - .prize-card-bg
  ├── 渐变背景：linear-gradient(145deg, blue-start, blue-end)
  ├── 圆角阴影：box-shadow: 0 20rpx 40rpx rgba(102, 126, 234, 0.3)
  ├── 装饰元素：光泽效果 (.prize-card-shine)
  └── 装饰点：三个 .decoration-dot
  
第2层（增强层）：稀有度光效 - <rarity-effects>
  ├── enabled=true：根据 rarity 添加光效边框和动画
  └── enabled=false：完全透明（rarityClass=''），第1层硬编码样式显示
  
第3层（顶层）：烟花特效（可选）
  └── winAnimation='fireworks' + (epic/legendary)
      → 20粒子CSS动画在最上层播放
```

## 🔧 技术细节

### 关键代码路径

**1. result-modal.wxml**（4个位置都使用）：
```xml
<rarity-effects 
  rarity="{{drawResult.prize.rarity || 'common'}}" 
  enabled="{{rarityEffects}}"
>
  <view class="prize-card-bg">
    <!-- 硬编码样式保底 -->
  </view>
</rarity-effects>
```

**2. rarity-effects.ts**（空字符串保底）：
```typescript
rarityClass: enabled ? `rarity--${safeRarity}` : ''
```

**3. rarity-effects.scss**（默认透明容器）：
```scss
.rarity-effects {
  /* 无任何默认样式，只有添加 rarity--xxx class 才有效果 */
}
```

**4. result-modal.scss**（硬编码保底样式）：
```scss
.prize-card-bg {
  background: linear-gradient(145deg, $gradient-blue-start, $gradient-blue-end);
  border-radius: 28rpx;
  padding: 4rpx;
  box-shadow: 0 20rpx 40rpx rgba(102, 126, 234, 0.3);
}
```
