## 🎯 更新内容

## 📦 文件修改清单

### 1. `result-modal.json`
```json
{
  "component": true,
  "usingComponents": {
    "rarity-effects": "../rarity-effects/rarity-effects"  // ← 新增引用
  }
}
```

### 2. `result-modal.wxml`
在以下4个位置添加了 `<rarity-effects>` 包裹：

#### ✅ 单次抽奖 - Simple 默认模式
```xml
<rarity-effects 
  rarity="{{drawResult.prize.rarity || 'common'}}" 
  enabled="{{rarityEffects}}"
>
  <view class="prize-card-bg">
    <!-- 奖品卡片内容 -->
  </view>
</rarity-effects>
```

#### ✅ 单次抽奖 - Card Flip 翻转模式
```xml
<rarity-effects 
  rarity="{{drawResult.prize.rarity || 'common'}}" 
  enabled="{{rarityEffects && cardFlipped}}"  <!-- 翻开后才显示光效 -->
>
  <view class="card-flip-card">
    <!-- 卡牌内容 -->
  </view>
</rarity-effects>
```

#### ✅ 多连抽 - Card Flip 翻转模式
```xml
<rarity-effects 
  rarity="{{drawResult.prizes[multiDrawCurrentIndex].rarity || 'common'}}" 
  enabled="{{rarityEffects && cardFlipped}}"
>
  <!-- 当前揭晓的奖品卡片 -->
</rarity-effects>
```

#### ✅ 多连抽 - Simple 默认模式
```xml
<rarity-effects 
  rarity="{{drawResult.prizes[multiDrawCurrentIndex].rarity || 'common'}}" 
  enabled="{{rarityEffects}}"
>
  <!-- 当前揭晓的奖品卡片 -->
</rarity-effects>
```

## 🔧 技术细节

### 1. 卡牌翻转模式的智能控制
```xml
enabled="{{rarityEffects && cardFlipped}}"
```
- 卡牌未翻开时：不显示光效（保持悬念）
- 卡牌翻开后：显示光效（揭晓瞬间的震撼）

### 2. 兜底机制
```xml
rarity="{{drawResult.prize.rarity || 'common'}}"
```
- 后端返回 rarity 字段：使用对应光效
- 后端未返回：默认使用 common（蓝色边框）

### 3. 开关控制
```xml
enabled="{{rarityEffects}}"
```
- 运营可通过后端配置 `display.rarity_effects_enabled` 控制
- true：显示稀有度光效
- false：不显示光效（保持简洁）

   ```json

       { "name": "500积分", "rarity": "rare" },
       { "name": "八八折券", "rarity": "epic" },
   ```
